# Cadastro de Usuários em Python

Projeto simples em Python para cadastrar e listar usuários no terminal.

Criado para treinar lógica de programação.


# Como usar:

Execute o arquivo no terminal ou no seu editor:

python cadastro.py


# Funcionalidades:

- Cadastrar usuário
- Listar usuários
- Menu interativo
- Sair do sistema


# Tecnologias: 

- Python 3
